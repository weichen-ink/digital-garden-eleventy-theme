# 更新日志

本文档记录 Digital Garden Eleventy Theme 的所有重要变更。

## [1.3.0] - 2025-11-02

### ✨ 新增功能

- **Favicon 支持**：添加可配置的网站图标功能，支持 `.ico`、`.svg`、`.png` 和 `site.webmanifest` 多种格式
- **动态端口显示**：开发服务器地址支持动态端口获取，当默认端口被占用时自动显示实际端口
- **完善的 Favicon 文档**：在 README 和主题配置文档中添加详细的 Favicon 配置说明

### ⚡ 性能优化

- **预览窗口优化**：
  - 移除防抖机制，使用更精确的状态跟踪
  - 新增 `pendingLink` 状态管理，避免状态混乱
  - 优化定时器管理逻辑，提升响应速度
  - 显著提升侧边栏反向链接卡片的预览稳定性

### 🐛 问题修复

- **反向链接摘录优化**：修复双链优先级问题，确保摘录优先显示双链标记内容而非普通文本
- **静态资源 404 错误**：修复 favicon、图片、字体等静态资源路径配置错误导致的 404 问题
- **预览窗口稳定性**：
  - 修复侧边栏中间卡片预览经常失败的问题
  - 修复鼠标在卡片内部移动时预览闪烁的问题
  - 修复快速移动鼠标时事件丢失导致的预览失效

### 📝 文档改进

- 添加 Favicon 配置详细说明（支持的格式、推荐尺寸、文件结构等）
- 优化开发服务器配置说明
- 改进主题配置文档的可读性

## [1.2.0] - 2025-10-30

### ✨ 新增功能

- **页面别名功能**：支持为页面配置多个访问路径，自动生成 SEO 友好的重定向页面
- **改进的 footer 显示**：优化页脚主题信息展示
- **演示站点更新**：更新到新的 Vercel 部署地址

### 🐛 问题修复

- 修复首页反向链接悬浮预览失效问题
- 修复 git 更改检测逻辑
- 简化 Vercel 配置，修复部署问题

### 📝 文档改进

- 完善文档结构，添加详细文档链接
- 简化 macOS 权限设置说明
- 更新演示链接位置

### ♻️ 代码重构

- 性能优化与代码重构
- 简化同步脚本，支持跨平台
